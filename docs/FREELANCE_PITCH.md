# 営業用サマリー（CrowdWorks / Coconala 提案用）

## このシステムで実現できること

✅ 小規模クリニック・調剤薬局の**業務効率化**  
✅ **既存のExcel管理からの脱却**  
✅ スマートフォン・PCどちらからでもアクセス可能  
✅ **導入コスト最小化**（月額数千円〜のSaaS費用不要）

## 標準搭載機能（カスタマイズ可能）

### すぐに使える機能
- データの登録・編集・削除
- ステータス管理（未処理→進行中→完了）
- 担当者・顧客のマスタ管理
- CSV/Excelでのデータ出力
- PDF帳票の自動生成
- メール自動通知

### オプション追加可能
- LINE通知対応
- QRコードスキャン
- 電子署名
- 外部システム連携（レセコン等）

## 技術スペック（信頼性の根拠）

- **テスト**: 単体テスト74件・E2Eテスト12件（自動化済み）
- **CI/CD**: GitHub Actions による自動テスト・デプロイ
- **監視**: Sentry によるエラー自動検知
- **セキュリティ**: JWT認証・ロールベースアクセス制御
